# StarZap — Site vitrine

Site web de présentation de **StarZap**, l'application mobile développée par **Green Island Tech**. L'appli qui libère ta voix : une expérience de découverte musicale gamifiée.

## Structure du projet

```
starzap/
├── index.html            # Page principale
├── css/
│   └── style.css         # Feuille de style (DA violet + vert néon)
├── js/
│   └── script.js         # Animations au scroll + comportement nav
├── assets/
│   ├── logo.png          # Logo officiel StarZap (fond transparent)
│   └── favicon.png       # Favicon navigateur
├── .vscode/
│   └── settings.json
└── README.md
```

## Ouvrir dans VS Code

1. `Fichier > Ouvrir le dossier…` → sélectionne `starzap`
2. Installe l'extension **Live Server** (Ritwick Dey)
3. Clic droit sur `index.html` → `Open with Live Server`

### Si l'explorateur VS Code n'affiche pas les fichiers

Clic droit sur le dossier racine dans l'explorateur → **Refresh** (ou F5). OneDrive peut mettre quelques secondes à synchroniser.

## Remplacer le logo

Le logo officiel est dans `assets/logo.png`. Pour le remplacer :
- Glisse ton nouveau PNG dans `assets/` et renomme-le `logo.png` (ça écrase l'ancien)
- Aucune modification du code nécessaire

## Direction artistique

- **Violet profond** : `#1A0033` → `#6B2FD9`
- **Rose néon** : `#D89EFA` → `#F5D4FF`
- **Vert lime** : `#C6FF3D` (accents, CTA)
- **Typos** : Orbitron + Space Grotesk
